MariaDB

Para acceder de forma remota a MariaDB debemos de tener nuestro PC con Debian, ademas de tener MariaDB instalado en nuestro dispositivo

PASOS PARA INSTALAR MARIADB

Lo primero que debemos hacer es actualizar con : apt update

Luego de esto debemos de instalar MariaDB con apt install mariadb-server

ADMITIR CONEXIONES EXTERIORES

Cuando tenemos instalado MariaDB, tendremos el siguiente archivo: /etc/mysql/50-server.cnf

Ese archivo accederemos para modificarlo con vi de la siguiente forma: vi etc/mysql/50-server.cnf

Dentro debemos de comentar una linea que viene dentro de la siguiente forma: #bind-address = 127.0.0.1

Una vez hecho lo anterior, debemos de reiniciar el servicio con: systemctl restart mariadb

USUARIO PARA ACCESO REMOTO

Para crear un usuario remoto primero debemos de acceder a MySQL, para ello realizaremos lo siguiente: mysql -u root y estaremos dentro

Aqui empezaremos de declarar un usuario remoto de la siguiente manera: CREATE USER 'nombre'@'%' IDENTIFIED BY 'nombre'

Con esto habremos creado nuestro usuario remoto para que se acceda desde fuera. Se establece las contrase√±as de la siguiente forma: SET PASSWORD FOR 'user-name'@'%' = PASSWORD('NEW_USER_PASSWORD'); FLUSH PRIVILEGES;

CONCEDER PERMISOS AL USUARIO

Para dar permisos a un usuario se sigue la siguiente estructura: GRANT ALL PRIVILEGES ON basedatos.* TO 'nombreU'@'%' IDENTIFIED BY 'nombreU';

Y con esto hemos terminado :D
